## How to run project

Clone repo

```
https://github.com/Ptopic/Locastic-node-js-assigment.git
```

Cd into backend folder and install dependacies

```
cd backend
npm i
```

<br>

Start dev server

```
npm run start
```

<br>

Route for calling api

```
http://localhost:3001/api/blog/
```

<br>

Swagger ui route

```
http://localhost:3001/api-docs/
```
